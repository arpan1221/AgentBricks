# Movie Recommender Story Arc

Learn to build a production-grade movie recommendation system through 6 progressive bricks.

## Learning Objectives

- Understand event-driven architecture
- Build feature engineering pipelines
- Train recommendation models
- Deploy low-latency serving systems
- Implement monitoring and observability
- Orchestrate end-to-end ML pipelines

## Bricks

- **brick-01-data-collection/**: Event streaming and data collection
- **brick-02-feature-engineering/**: Feature computation and feature store
- **brick-03-model-training/**: Model training and model registry
- **brick-04-recommendation-service/**: Low-latency serving with retrieval and ranking
- **brick-05-monitoring/**: Observability, metrics, and alerting
- **brick-06-orchestration/**: Pipeline automation and scheduling
